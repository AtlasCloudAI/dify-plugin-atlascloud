## Overview

AtlasCloud is a cloud-based AI platform that provides high-performance model inference services. With an OpenAI-compatible API, AtlasCloud delivers fast and reliable access to popular large language models including Claude, GPT, Gemini, and more. Check all models [here](https://www.atlascloud.ai/models?utm_source=github&utm_medium=link&utm_campaign=dify_plugin_atlascloud).

This plugin seamlessly integrates AtlasCloud's models into Dify, enabling developers to leverage AtlasCloud's infrastructure for AI-powered applications.

## Configuration

1. [Login to Atlas Cloud](https://console.atlascloud.ai?utm_source=github&utm_medium=link&utm_campaign=dify_plugin_atlascloud) or [Create an account](https://www.atlascloud.ai/login?utm_source=github&utm_medium=link&utm_campaign=dify_plugin_atlascloud).
2. Open your [API keys](https://console.atlascloud.ai/settings?utm_source=github&utm_medium=link&utm_campaign=dify_plugin_atlascloud) page.
3. Select **Create API Key** to create an API key, optionally naming the key.
4. Configure in Dify: Open Settings â†’ Model Provider, find Atlas Cloud, and enter your API key in the API Key field.
5. Save: Click "Save" to activate the plugin.Now you can use the AtlasCloud AI models in Dify.
